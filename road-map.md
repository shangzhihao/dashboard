# React + Dashboard Roadmap (05.png)

Goal: Build a React dashboard that matches the layout/feel of `web_image/05.png` using Recharts (hover, click, brush zoom).

Day 1: React basics + project setup
- Learn: JSX, components, props, state, and the React project structure.
- Implement: Run the app, explore `src/App.js`, and render a simple header + sidebar skeleton.

Day 2: Layout + CSS fundamentals
- Learn: Flexbox, CSS Grid, spacing, and responsive breakpoints.
- Implement: Build the top bar, left sidebar, and main content grid to match the screenshot layout.

Day 3: Navigation pills + filters
- Learn: Mapping arrays to UI, basic event handling.
- Implement: Add the pill navigation row and the filter dropdowns + query button.

Day 4: First chart with Recharts
- Learn: Recharts components, `ResponsiveContainer`, `LineChart`/`ComposedChart`.
- Implement: Create the main line chart with mock data and hover tooltips.

Day 5: Interactions
- Learn: Tooltip customization, click handlers, Brush slider.
- Implement: Add custom tooltip styling, click logging, and brush zoom.

Day 6: Visual polish
- Learn: Design tokens, theming with CSS variables.
- Implement: Refine colors, typography, spacing, and shadows to match the feel of 05.png.

Day 7: Responsiveness + cleanup
- Learn: Media queries and mobile layout basics.
- Implement: Make layout stable on tablet/mobile and tidy component structure.

Day 8: Data wiring (optional)
- Learn: Fetching data, handling async state.
- Implement: Replace mock data with API data (if available).

Day 9: Final review
- Learn: Simple QA checklist.
- Implement: Fix UI bugs, align sizes, and verify chart behavior.

Day 10: Deploy (optional)
- Learn: Build + deploy basics.
- Implement: Deploy to Vercel or Netlify.
